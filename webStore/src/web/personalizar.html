Personalizar su tienda en línea

Esta aplicación crea la tienda en línea para el código de cliente (AD_Client_ID) número 11 (GardenWorld)

Para personalizar la aplicación, necesita:

  • Identifíquese como administrador del cliente:
    • En la ventana Cliente, cambie el directorio web, por ejemplo a XXX.
    • Haga click en la esquina inferior derecha de la ventana para conseguir su código de cliente (AD_Client_ID), por ejemplo 1000000.
  • Descomprima el archivo openXpertyaWebStore.war
  • En el directorio WEB_INF:
    • Edite el archivo web.xml para cambiar el código de cliente (AD_Client_ID) que aparece en pantalla por su propio código de cliente, por ejemplo 1000000.
    • Edite el archivo jboss-web.xml y configure su directorio web, por ejemplo XXX.
  • Cambie el UI de los archivos HTML y JSP si se requiere.
  • Vuelva a comprimirlo con otro nombre, por ejemplo myStore.war antes de copiarlo a $OPENXPERTYA_HOME/jboss/server/openXpertya/deploy.
