Document Sets

JavaService documentation is included in html format as part of the software download distribution.

If you are reading this from your local drive, then you already have this documentation!

The documentation is split to cover different areas of interest, for 'end users' and 'developers' making use of the JavaService software.

User Documentation

User documentation describes how the software is seen by the end user when used in a Windows environment.
It lists the commands available from the utility for installing, querying, stopping, starting and removing a Java application as a service under Windows NT.

Developer Documentation

Developer documentation is intended for technical users who may be developing a Java application to be run as a service.
It specifies the lower-level mechanisms used by the software and defines the application interface that it uses, plus the integration with features of the Microsoft Windows environment.

Older Documents

Version 1.2 documentation is included for reference only, being a copy of the older document set provided with earler (pre-V2) releases of JavaService.
