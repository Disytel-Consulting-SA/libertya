JdbcRowSet Sample

README


    This document presents a walkthrough of the JdbcRowSet Sample, it explains the JdbcRowSet sample provided using snippets
of code.

    Overview 

        This is sample to demonstrate the functionality of a JdbcRowSet. This is like a ResultSet but behaves like a Java Bean component
   providing the getters and setters allowing it to be used in cases where Java Beans functionality is required. The example demonstartes
   the operations that can be performed with this component which is simlar to a ResultSet.
  
   Project RAVE is using this as a Java Beans component in their tool.


    Explanation of the Sample


        // Get the data into memory ,using populate
      // This JdbcRowSet behaves exactly like the ResultSet
      // whose handle has been passed to it.
         
      jdbcrs = new JdbcRowSetImpl(dbUrl,dbUserId,dbPasswd);
      jdbcrs.setCommand(dbCommand);
      jdbcrs.execute();

         The above code creates a JdbcRowSet using the constructor that takes the Database URL, Username and password as the parameters.
       The setCommand function is sued to sepcify the command that is to be used to poulate the JdbcRowSet. Then the execute method is
       invoked to populate the JdbcRowSet.
         
        
          // Now we have a hnadle to a JdbcRowSet.
      // This is similar to a ResultSet but the advantage being
      // it implementation of the RowSet interface making it
      // behave like a Java Beans component.
      // We can do all the operations that can be performed using
      // a ResultSet andle.
         
      jdbcrs.next();        
      jdbcrs.updateInt(1,9999);

          The above code shows the updation of a column ona particular row. Once the curosr is on the required row updateXXX functions can be called
         to update the corresponding columns. The changes need not be synced back to the Database explicitly as this is a connected type of RowSet
        and has a connection handle to the Database.
         
          // Similarly the values in other columns can also be updated
      // and the values written back to the DB.                
      jdbcrs.close();

        This will close the JdbcRowSet and release all the resource that were held by this object. This is a necessary operation that is to be done when
         no further operations are to be performed on that object so that memory leaks are avoided.
