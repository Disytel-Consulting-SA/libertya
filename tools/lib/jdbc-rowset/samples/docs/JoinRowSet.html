JoinRowSet Sample

README


        This document presents a walkthrough of the JoinRowSet Sample, it explains the JoinRowSet sample provided using snippets
of code.


    Overview

        This is a sample to demonstrate the functionality of a JoinRowSet. This example shows how to perform an in memory join of
   two already fetched rowsets. The user can set the join columns and just add the pre fetched rowsets to a JoinRowSet which
   performs the join. This is very useful in cases where data is present in two different datasources and need to be joined to form
   a relation.


    Explanation of the Sample

        // Instantiating the JoinRowSet
      jrs = new JoinRowSetImpl();

          The above command instantiates a new JoinRowSet that is empty and can be used to perform the join on the RowSets.
         
          // Setup the properties for the CachedRowSet
      crs1 = new CachedRowSetImpl();
      crs1.setUrl(dbUrl);
      crs1.setUsername(dbUserId);
      crs1.setPassword(dbPasswd);         
         
      Connection con = DriverManager.getConnection(dbUrl,dbUserId,dbPasswd);
      Statement stmt = con.createStatement();
      ResultSet rs = stmt.executeQuery(dbCommand1);        
      
      // Get the data into memory ,using populate
      crs1.populate(rs);

           The above piece of code is similar to that piece of code in the cachedRowSet sample that is used to poulate a
         CachedRowSet that will participaet in the join. For this populated CachedRowSet the match column needs to
         be set so that the join is to be performed on this column. Now add this rowset to the JoinRowSet , this is the
         first RowSet participating in the join.
                          
           crs1.setMatchColumn(4);
       jrs.addRowSet(crs1);
         
       // Setup the properties for the CachedRowSet
       crs2 = new CachedRowSetImpl();
       crs2.setUrl(dbUrl);
       crs2.setUsername(dbUserId);
       crs2.setPassword(dbPasswd);
                   
       ResultSet rs1 = stmt.executeQuery(dbCommand2);        
      
       // Get the data into memory ,using populate
       crs2.populate(rs1);                 
       crs2.setMatchColumn(2);
       jrs.addRowSet(crs2);
        The above snippet is similar to the first part except that here the second RowSet that will participate in the join is
        populated and the macth column set and added to the JoinRowSet so that join has now bwwn formed.

        Note : JoinRowSet  currently supports only INNER_JOIN formation.
