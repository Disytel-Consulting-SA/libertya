Release 1.0.1

Details

The 1.0.1 release represents our JSR 114 Maintenance Release ship. The following fixes have been done in this release.
5053098 : RowSet API should handle special characters in XML files
5055528 : populate fails with SQLException because locatorsUpdateCopy is not implemented
5063259 : javax.sql.rowset.BaseRowSet: spec ambiguity
5063277 : JoinRowSet throws SQLException with invalid column index
5063284 : javax.sql.rowset.serial.SerialClob throws unexpected NPE
5067095 : SyncFactory.getRegisteredProvider() needs to return Enum of providers
5070450 : cachedrowset fails in Tomcat when datasource name is passed with "url cannot be null"
5073393 : JSR-114: Jdbc SyncFactory Null pointer exception on CachedRowset initialization
5075041 : Incomplete fix for SyncFactory.getRegisteredProvider()
