JDBCTM RowSet Implementations - JavaTM WSDP Co-Bundle Directory Structure

After installing this bundle by unzipping it over thw JWSDP 1.4 install directory, will find the JDBC RowSet Implementations components in the jdbc-rowset directory. This contents breaks down as follows: 
    jwsdp-1.4
        jdbc-rowset
            docs
                ri              <-- Contains the RI Javadoc
                spec            <-- Contains the Specification
                tutorial        <-- Contains the JDBC RowSet Tutorial
            lib
                rowset.jar      <-- RI binary
            sample              <--             
                etc             <-- Contains configuation scripts for sample code
                webrowset-demo  <-- Contains a fully deployable WebRowSet application
                jdbcrowset      <-- JdbcRowSet sample application 
                cachedrowset    <-- CachedRowSet sample application
                webrowset       <-- WebRowSet sample application
                filteredrowset  <-- FilteredRowSet sample application
                joinrowset      <-- JoinRowSet sample application

JavaTM 2 Platform Standard Edition (J2SETM) Requirements

This release of the JDBC RowSet Implementations RI requires J2SE SDK, version 1.4 or higher.

JDBC Driver Requirements

Any JDBC driver used for connectivity with a SQL datasource must be a JDBC 3.0 compliant driver.
