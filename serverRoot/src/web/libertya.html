Existen muchas maneras de trabajar con Libertya. Actualmente disponemos de clientes para virtualmente cualquier sistema operativo. Para simplificar el acceso a los nuevos usuarios, todas las posibilidades aparecen enlazadas a continuación. En el menú de la izquierda dispone de enlaces adicionales a aquellas herramientas que podría necesitar instalar en su máquina, así como de los enlaces para que encuentre el soporte adecuado o nos lo solicite, y enlaces a herramientas de control y gestión de su servidor de aplicaciones.

Acceso a Libertya mediante el cliente web, sin necesidad de descargar ningún archivo.

Acceso al cliente de escritorio mediante Libertya Start para Linux. Requiere contar con wget. En la mayoría de distribuciones de Linux ya viene instalado por defecto.

Acceso al cliente de escritorio mediante Libertya Start para Windows. Requiere contar con wget para Windows instalado y configurado en el PATH.

Acceso al cliente de escritorio mediante Java Web Start. Requiere versión de Java con soporte para JNLP.
